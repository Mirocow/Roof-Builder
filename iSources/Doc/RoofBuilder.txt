Попытки сделать расчет самостоятельно без необходимой базы знаний обычно приводят 
к конфликтам и ошибкам, самой неприятной из которых является недобор, когда требуется 
срочным порядком докупать недостающие материалы. Чтоб избежать подобных тяжб, 
была создана специализированная программа для расчета кровельных покрытий, 
в последствии была модернизированна и в настоящий момент предствляет собой удобное средство 
для расчета и составления визуализированных карт раскроя. Преимущественно компьютерная программа Roof Builder 
создавалась для расчета потребности кровельного материала с представлением схемы укладки листов 
на каждом элементе кровли.

Внешний вид:
1) Изменен внешний вид главного окна проекта.
2) Доработан менеджер проектов (сортировка), улучшена возможность выбора текущей папки.
3) В менеджере проекта добавленна возможность отображения состояния. (те появился индикатор показывающий, рассчитана плоскоть или только нарисованна)
4) Упращено окно прорисовки плоскостей.
5) Во всех окнах, где это нужно, добавлен элимент. Он позволяет оптимизировать рабочее пространство и представляет из себя раздилитель окна, который можно перемещать и при этом увеличивать или уменьшать рабочую область.
6) Изменен вневний вид окна установки свойств программы 

Юзабилити:
1) В менеджере проектов добавленна сортировка и поиск по датам создания проектов.
2) Добавлен в менеджер проекта автоматический пересчет потребности материала, для неограниченного числа выбранных плоскостей.
3) В главном окне проекта, добавлена возможность сортировки профилей по фирмам их производителей.
4) Обновленна инструкция по применению.
5) Добавлен блок "Клиенты". Он ориентирован на хранение и вывод информации по клиентам, для которых производился рассчет.
6) Добавлена возможность запоминания и изменения точек начала рассчета (для автоматического режима). Устанавливается в окне свойств прогаммы\Настройки расчетного модуля.
7) Теперь в главном окне возможно будет изменять тип проекта с RFD на RBP.
8) Теперь почти все операции можно произвести, не выходя из менеджера проекта. Такие как: Копирование, Зеркальное копирование, Очистка, Автоматический расчет, Добавление комментарий, для выбранных поверхностей.

Расчет и печать:
1) Расчет, вывод на печать и сохранение комплектации.
2) Добавленна возможность рассчета стоимости материала и вывода спецификации на печать.

Движок:
1) Вся конфигурация для рассчетов хранится в открытой базе данных в формате Microsoft Access MDB.
2) Обновлен формат данных *.rbp - только для версии Prof.
3) Исправлены многочисленные ошибки (читайте истою изменений в программе).
4) Локализация: Russian, English.
5) Переделаны внутренности программы, для обеспечения наилучшей производительности.
6) Установленна и налажена система контроля лицензий и обновления компонентов программы через интернет. Кодовое название RDS. Registration Download Service - подробнее читайте здесь.
7) Добавленна возможность заносить неограниченное количество групп профилей а также наименований их производителей.
